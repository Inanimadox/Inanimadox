MG400

1Ultracompacto

El MG400 es un brazo robótico de escritorio ultracompacto con un tamaño más pequeño que una hoja de papel de tamaño A4.

2Brazo Colaborativo

Es seguro para la colaboración humana, fácil de usar y rápido de implementar, lo que lo hace perfecto para automatizar producciones flexibles de lotes pequeños.

3Caracteristicas del MG400

Con una carga útil máxima de 750 g, un alcance máximo de 440 mm, características de detección de colisión y arrastrar para enseñar, el MG400 es una opción rentable para tareas de automatización inteligente.

  

Magician

1Movilidad Omnidireccional

El Magician Go está equipado con ruedas mecanum para permitir movimientos laterales, diagonales y giratorios sin dirigir las ruedas, lo que hace posible las operaciones en espacios reducidos.

2Navegación inteligente

El Magician Go tiene un odómetro incorporado para la transmisión de datos de posición en tiempo real. Con la simulación del entorno dinámico sintético, los usuarios pueden monitorear fácilmente el movimiento del robot y planificar rutas de viaje.

3Conducción Autónoma con IA

El algoritmo de IA desplegado localmente realiza reconocimiento de imágenes, texto y rostro sin conexión a la red para permitir el reconocimiento de carreteras, la detección de letreros de calles, la detección de peatones y la operación del brazo robótico mientras se conduce.
